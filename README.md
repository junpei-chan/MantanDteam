## アプリケーション概要
チーム制作授業の課題として、以下のテーマに沿って制作を進めました。

- 架空のファミリーレストラン『マンタン』のタブレット注文画面の設計・制作
- ユーザー（お客様）の立場に立ち、「見やすく」「使いやすい」注文体験を考えて形にしていくことが目的です
